# Meteor Framework v3 Agent Skills

This repository contains a collection of skills for AI agents, designed to assist with full-stack development using the Meteor.js framework (v3.x and later).

## Available Skills

-   **`meteor-fullstack`**: Provides comprehensive knowledge for full-stack Meteor 3.x development, including React, MongoDB, async APIs, methods, pub/sub, and GraphQL patterns.

## Installation

To add these skills to your agent, run the following command:

```bash
npx skills add kolyasya/meteor-toolbox
```
